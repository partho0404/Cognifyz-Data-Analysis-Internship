#  Data Analysis Internship Project

This repository contains my work completed during the **Data Analysis Internship** at Cognifyz Technologies.

The project focuses on analyzing a restaurant dataset to extract meaningful insights using Python and data visualization techniques.

---

##  Project Overview

The analysis includes:

###  Level 1 Tasks
- Top cuisines analysis  
- City-wise restaurant analysis  
- Price range distribution  
- Online delivery impact on ratings  

###  Level 2 Tasks
- Restaurant rating distribution  
- Cuisine combination analysis  
- Geographic analysis (location clustering)  
- Restaurant chain analysis  

---

##  Tools & Technologies Used

- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

##  Key Insights

- Certain cuisines dominate customer preferences  
- Most restaurants fall in the moderate price range  
- Online delivery positively influences ratings  
- Restaurants are clustered in urban areas  
- Popular chains do not always have the highest ratings
